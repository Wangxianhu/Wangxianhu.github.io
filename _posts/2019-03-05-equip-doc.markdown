## **<a href='#ml' name='aa1' >1\. 属性</a>**

### 接口功能

根据设备类型查询与类型相关的产品数据

### URL

[http://39.106.198.108:8140/cover-web/sXShow](http://39.106.198.108:8140/cover-web/sXShow)

### 支持格式

json

### HTTP请求方式

post

### 请求参数

|参数|必选|类型|说明|
|:-----  |:-------|:-----|-----|
|id|yes|数字|设备的id|
|sblx|yes|数字|设备类型0井盖，1终端，2蓝牙锁|

### 返回字段

均是中文字段

### 接口示例
